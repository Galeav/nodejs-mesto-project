Ошибки в маршрутах:
- 400 - переданы некорректные данные
- 401 - требуется авторизация (`/signin`) или введенные логин + пароль на `/signin` неверные
- 403 - при попытке удалить карточку, когда `req.user` не является её владельцем (если поменять `_id` во временном решении авторизации)
- 404 - карточка / пользователь не найден
- 409 - введённый при регистрации (`/signup`) email уже есть в БД
- 500 - другие ошибки

Тест для `urlRegex` (поддерживает кирилицу для .рф-доменов):
`npm run test-regex`

Запуск без сборка:
`npm run start`

Запуск с горячей перезагрузкой:
`npm run dev`

Сборка:
`npm run build`

Проверка линтером:
`npm run lint`
